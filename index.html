ヘルプ

操作説明

フォルダを開く
レビュー対象の Markdown があるフォルダを選択します。コメントはフォルダ内の .local-markdown-reviewer に自動保存されます。
Git branch 表示
選択したフォルダが Git リポジトリの場合、上部に現在の branch を表示します。
コメント追加
本文の行番号をクリック、または本文を選択してからコメントを追加します。
対象へ移動
コメント一覧から対象行へ移動します。編集モード中は編集欄の対象行を選択します。
返信と解決
コメントには返信を追加できます。対応済みになったものは解決済みに変更できます。
本文編集
編集ボタンで Markdown 本文を直接編集できます。保存すると表示とコメント位置を再計算します。
目次と表示設定
見出しがあるファイルでは目次を表示できます。表示サイズ、行番号、コメント表示はその他メニューから変更できます。
