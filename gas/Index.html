ヘルプ

操作説明

フォルダ選択
Google Drive の Markdown フォルダを選択します。共有アイテムは検索から探せます。
IDから読み込み
Drive folder ID を直接入力して読み込めます。ID入力はその他メニュー内にあります。
コメント追加
本文の行番号をクリック、または本文を選択してからコメントを追加します。
対象へ移動
コメント一覧から対象行へ移動します。編集モード中は編集欄の対象行を選択します。
返信と解決
コメントには返信を追加できます。対応済みになったものは解決済みに変更できます。
本文編集
編集ボタンで Markdown 本文を直接編集できます。保存すると Drive 上のファイルを更新します。
目次と表示設定
見出しがあるファイルでは目次を表示できます。表示サイズ、行番号、コメント表示はその他メニューから変更できます。
